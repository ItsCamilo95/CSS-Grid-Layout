CSS Grid Layout

Es un sistema de maquetación que permite crear diseños bidimensionales para una página web. Permite dividir la página en áreas principales, definir tamaño, posición y orden de los elementos. También permite adaptar el diseño a diferentes tipos de pantalla.


Sus principales ventajas son:

  • Es un estándar de CSS que no necesita librerías externas ni clases adicionales.
  • Ofrece un control total de las dos dimensiones (filas y columnas).
  • Es responsive y adaptable a los dispositivos.
